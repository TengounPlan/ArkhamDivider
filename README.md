# Arkham Divider: Divider Constructor App

https://neizerth.github.io/ArkhamDivider

## Installation

1. create `.env.local` in project root with

```txt
VITE_BASE_URL="https://neizerth.github.io/ArkhamDividerData"
```

2. install dependencies

```bash
npm install
```

3. run locally

```bash
npm run dev
```
